# Zerto Automation with Ansible & Semaphore UI

Welcome to the Zerto Automation project! This repository contains automation workflows for Zerto disaster recovery operations.

## 📋 Project Overview

This project automates Zerto operations using Ansible playbooks, orchestrated through Semaphore UI. It provides a modular, repeatable, and UI-driven workflow for DR automation.

- ✅ Ansible-driven Orchestration

## 🛠️ Getting Started

### Prerequisites
- Python 3.8+
- Ansible 2.9+
- Zerto API credentials
- Semaphore UI instance

### Quick Start
```bash
git clone https://github.com/adobao/Automation.git
cd Automation
pip install -r requirements.txt
ansible-playbook playbooks/check_vpg_status.yml
```
