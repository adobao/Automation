📋 Project Overview

This project automates Zerto operations using Ansible playbooks, orchestrated through Semaphore UI. It provides a modular, repeatable, and UI-driven workflow for disaster recovery automation.

Whether you need to perform failovers, failbacks, or regular VPG status checks, this automation framework makes it simple, reliable, and repeatable.

🚀 Getting Started

Prerequisites

  • Python 3.8 or higher
  • Ansible 2.9 or higher
  • Zerto API credentials
  • Semaphore UI instance running
  • Git installed

Installation Steps

1. Clone the Repository

git clone https://github.com/adobao/Automation.git cd Automation

2. Install Python Dependencies

pip install -r requirements.txt

3. Configure Ansible Inventory

Edit inventory/hosts with your Zerto servers and update group variables in inventory/group_vars/

4. Import Semaphore Templates

Log into your Semaphore UI instance and import playbooks from the /playbooks/ directory

5. Run Your First Automation

ansible-playbook playbooks/check_vpg_status.yml -i inventory/hosts

💡 Usage Examples

Check VPG Status

Check the status of all Virtual Protection Groups (VPGs):

ansible-playbook playbooks/check_vpg_status.yml -i inventory/hosts

Perform Failover

Initiate a failover for specified VPGs:

ansible-playbook playbooks/failover.yml -i inventory/hosts --extra-vars "vpg_name=MyVPG"

Perform Failback

Execute a failback operation:

ansible-playbook playbooks/failback.yml -i inventory/hosts --extra-vars "vpg_name=MyVPG"

Using Semaphore UI

  1. Log into Semaphore UI at http://your-semaphore-instance:3000
  2. Select your project
  3. Choose a template (e.g., "Check VPG Status")
  4. Click "Run"
  5. Monitor execution in real-time
  6. View logs and results
